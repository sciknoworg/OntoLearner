Learner Pipeline
====================

``LearnerPipeline`` supports:

- retriever-only mode (set ``retriever_id``)
- llm-only mode (set ``llm_id``)
- rag mode (set both ``retriever_id`` and ``llm_id``), or provide a prebuilt ``rag`` learner
